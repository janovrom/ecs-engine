**Key Insight**: Pre-computed at compile time avoids reflection on every `GetOrRegister<T>()` call.

### ComponentTypeRegistry Fast Path

```csharp
[MethodImpl(MethodImplOptions.AggressiveInlining)]
public ComponentTypeInfo GetOrRegister<T>() where T : struct, IEcsComponent
{
    var type = typeof(T);
    
    // Fast path: generated registry
    if (_GeneratedRegistry.TryGetValue(type, out var generated))
        return generated;
    
    // Fallback: dynamic registration (edge case)
    return GetOrRegisterDynamic<T>(type);
}
```

**Performance**: O(1) hash lookup instead of reflection-based attribute scanning.
